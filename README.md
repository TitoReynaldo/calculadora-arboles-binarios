# Binary Tree Expression Calculator

## Arquitectura del Proyecto
Implementación de un evaluador de expresiones matemáticas utilizando la estructura de datos de Árboles Binarios (Binary Trees). El sistema modela operaciones algebraicas asignando operadores a los nodos internos y operandos a los nodos hoja, resolviendo la ecuación mediante un recorrido recursivo (Post-order traversal).

## Especificaciones Técnicas
* **Lenguaje:** Python 3.x
* **Estructuras de Datos:** Nodos, Árboles Binarios (AST - Abstract Syntax Tree).
* **Dependencias:** Librería estándar nativa de Python. Tolerante a fallos sin requerir `pip install`.
